Zenn.devの記事ページを参考にした改善計画です。

## Phase 1: 目次の機能強化（優先度: 高）

### 1.1 スクロール連動の現在位置ハイライト
**概要**: 記事をスクロールすると、現在読んでいるセクションが目次でハイライトされる

**実装方法**:
- Intersection Observer APIを使用
- 現在のセクションの目次項目にアクティブクラスを付与
- スムーズなトランジションでハイライト

**参考**: Zennでは現在位置の項目が青色でハイライトされる

**変更ファイル**:
- `src/pages/blog/[id].astro` - React Islandとして実装
- `src/components/TableOfContents.tsx` - 新規作成
- `src/styles/main.scss` - アクティブ状態のスタイル追加

---

## Phase 2: 左サイドバー（リアクションバー）（優先度: 高）

### 2.1 固定リアクションバーの追加
**概要**: 記事の左側にスクロール追従する固定のリアクションバーを配置

**含める要素**:
- いいねボタン（カウント表示）
- ブックマークボタン（カウント表示）
- シェアボタン群
  - X（Twitter）
  - Facebook
  - LinkedIn
  - リンクコピー

**実装方法**:
- `position: sticky` で固定
- 現在の記事下部のリアクション/シェアボタンを移動
- モバイルでは非表示（既存の記事下部UIを使用）

**参考**: Zennでは左側に縦並びで固定表示

**変更ファイル**:
- `src/pages/blog/[id].astro` - レイアウト調整
- `src/components/StickyReactionBar.tsx` - 新規作成
- `src/styles/main.scss` - スタイル追加

---

## Phase 3: 右サイドバーの強化（優先度: 中）

### 3.1 著者情報カードの追加
**概要**: 目次の上に著者情報カードを表示

**含める要素**:
- アバター画像
- 著者名
- 短い自己紹介
- フォローボタン（SNSリンク）
- 「バッジを贈る」的な機能（オプション）

**実装方法**:
- microCMSのprofileデータを使用
- 目次コンテナの上に配置

**参考**: Zennでは著者カードが目次の上に表示

**変更ファイル**:
- `src/pages/blog/[id].astro` - 著者カード追加
- `src/components/AuthorCard.astro` - 新規作成

---

## Phase 4: 記事ヘッダーの改善（優先度: 中）

### 4.1 記事アイコン（絵文字）の追加
**概要**: 記事タイトルの上に大きな絵文字アイコンを表示

**実装方法**:
- microCMSに`emoji`フィールドを追加
- タイトル上部に中央揃えで表示
- デフォルト絵文字を設定（未設定時）

**参考**: Zennでは記事ごとに絵文字アイコンが設定可能

### 4.2 タグのスタイル改善
**概要**: カテゴリ/タグをカラフルなバッジスタイルに

**実装方法**:
- カテゴリごとに色を設定
- アイコン付きバッジデザイン
- ホバーエフェクト

**参考**: Zennではタグがカラフルなバッジで表示

**変更ファイル**:
- `src/pages/blog/[id].astro` - ヘッダー部分の調整
- `src/styles/main.scss` - タグスタイル追加

---

## Phase 5: 記事本文のスタイリング（優先度: 中）

### 5.1 リンクカードプレビュー
**概要**: URLを貼ると自動でOGP情報を取得してカード表示

**実装方法**:
- ビルド時にOGP情報を取得
- カードコンポーネントで表示
- サムネイル、タイトル、説明、ファビコン

**参考**: Zennでは外部リンクがカード形式で表示

### 5.2 コードブロックの改善
**概要**: シンタックスハイライトの強化

**実装方法**:
- Shikiの導入（現在のPrismから移行）
- ファイル名表示
- コピーボタン
- 行番号表示（オプション）

**参考**: Zennは最近Shikiに移行

### 5.3 埋め込みコンテンツの対応
**概要**: Twitter/X、GitHub Gist等の埋め込み対応

**実装方法**:
- Twitter埋め込みウィジェット
- GitHub Gist埋め込み
- YouTube埋め込み

**変更ファイル**:
- `src/lib/` - OGP取得関数追加
- `src/components/LinkCard.astro` - 新規作成
- `src/components/EmbedTwitter.tsx` - 新規作成

---

## Phase 6: 読書体験の向上（優先度: 低）

### 6.1 読了プログレスバー
**概要**: 記事の読了進捗を画面上部にバーで表示

**実装方法**:
- スクロール位置から進捗を計算
- 画面上部に細いプログレスバー表示
- シアン色のグラデーション

### 6.2 推定読了時間の改善
**概要**: より正確な読了時間の計算

**実装方法**:
- 日本語文字数を考慮（400文字/分程度）
- コードブロックの時間も考慮

### 6.3 前後の記事ナビゲーション
**概要**: 記事下部に前後の記事へのリンクを追加

**実装方法**:
- 同カテゴリ内での前後
- または公開日順での前後

**変更ファイル**:
- `src/components/ReadingProgress.tsx` - 新規作成
- `src/pages/blog/[id].astro` - ナビゲーション追加

---

## Phase 7: アクセシビリティとパフォーマンス（優先度: 低）

### 7.1 キーボードナビゲーション
**概要**: キーボードでの記事操作を改善

**実装方法**:
- `j`/`k`キーでセクション間移動
- `Escape`キーでトップへスクロール

### 7.2 画像の遅延読み込み最適化
**概要**: 画像のLazy Loading改善

**実装方法**:
- Intersection Observerでの遅延読み込み
- プレースホルダー画像の改善（LQIP）
- 画像のアスペクト比予約
